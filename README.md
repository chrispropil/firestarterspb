# Firestarter SPB Research Sandbox

This repository is a research sandbox for Firestarter SPB (Structural Participation Breakdown) work.

## Purpose

This repo is separate from `Fast-Lane-Research` and is intended for isolated Binance/Bitget research replay experiments, data-source audits, and SPB hypothesis review.

## Boundaries

- No live trading.
- No trading signals.
- No Cell 2 promotion.
- No production automation.
- No API keys, credentials, or secrets.
- No mutation of live scanner data.
- No raw market data committed to the repository.

## Current First Research Lane

Stage 1 is a Binance 5-token / 1-month pilot for:

- SOLUSDT
- XRPUSDT
- DOGEUSDT
- LINKUSDT
- AVAXUSDT

Required research inputs include 1H candles, 4H candles, funding, open-interest statistics, current open interest, and long/short ratios where available.
