# Paid Data Source Evaluation: SPB & DNA Tape Integration

## Objective
Evaluate whether a single paid data vendor can unify the historical data requirements for **both** the Firestarter SPB (macro market-structure replay) and the future DNA/Tape project (microstructure, footprint, and order-flow studies).

## Primary Candidate Evaluated: CoinGlass API (V4)
**CoinGlass API** is highly specialized in crypto derivatives and market structure.

### Coverage Assessment
- **Historical OHLCV:** Yes. Excellent coverage across all major exchanges.
- **Historical Open Interest (OI):** Yes. Deep historical coverage, aggregated and per-exchange.
- **Historical Funding Rates:** Yes. Complete history for perpetual futures.
- **Liquidations:** Yes. Historical liquidation data and heatmaps natively available.
- **Long/Short Ratios:** Yes. Comprehensive (Top Trader, Global Account, Position ratios) directly accessible.
- **Taker Buy/Sell (Order Flow):** Yes. Explicitly tracks taker execution behavior.
- **L2/L3 Order Book & Footprint:** Yes. V4 API provides detailed order book depth (L2/L3) and footprint analytics (price-level execution imbalances).
- **Exchange/Symbol Filtering:** Yes. Highly granular filtering across 30+ exchanges.
- **6+ Months History:** Yes.
- **CSV / API Export:** Yes, robust REST API (JSON) that can be easily parsed to CSV.

**CoinGlass Verdict:** **Strong Candidate.** CoinGlass successfully bridges the gap between high-level SPB metrics (OI, Funding, L/S) and deep DNA/Tape microstructure metrics (Footprint, L2/L3, Taker flow).

---

## Comparison Candidate: Tardis.dev
**Tardis.dev** is the gold standard for high-frequency, tick-level microstructure research.

### Coverage Assessment
- **L2/L3 & Tick Data:** Exceptional. True tick-level trades and deep L2/L3 order book history.
- **Taker Buy/Sell:** Yes. Derived flawlessly from raw tick trades.
- **Open Interest & Funding:** Yes. Historically archived (often generated by polling exchange REST endpoints).
- **Long/Short Ratios:** **Fragmented/Partial.** Tardis archives what the exchange natively broadcasts. While Binance top-trader ratios and Huobi elite ratios are available, it requires significantly more manual data engineering to normalize across exchanges compared to CoinGlass.
- **Liquidations:** Yes.

**Tardis Verdict:** **Specialized for DNA/Tape.** Tardis is unmatched for pure microstructure (DNA), but using it for SPB requires heavy manual aggregation for higher-level metrics like L/S ratios.

---

## Comparison Candidate: CCData (CoinDesk Data)
**CCData** provides standard, institutional-grade market data.

### Coverage Assessment
- **OHLCV & Order Book:** Solid historical coverage for standard pricing and L2 depth.
- **Derivatives Metrics:** Supports OI and funding, but generally lacks the advanced retail/trader positioning insights (Footprint, Liquidation Heatmaps, hyper-granular L/S ratios) that are crucial for Firestarter's specific edge.

**CCData Verdict:** **Insufficient for Firestarter Edge.** Too generalized; lacks the deep derivatives footprint required for DNA/Tape.

---

## Final Evaluation & Recommendation

**Recommendation:** **Select CoinGlass API (V4).** 

CoinGlass is the only candidate that naturally provides both the structured SPB derivatives data (OI, L/S Ratios, Liquidations) and the DNA/Tape microstructure data (Footprint, L2/L3 Depth) out of the box in a unified API, saving massive amounts of data engineering time.

If extreme high-frequency tick-level replay becomes the primary focus for DNA in the distant future, Tardis can be added strictly as a micro-supplement. However, for current unified research bridging SPB to DNA, CoinGlass covers the complete stack.
